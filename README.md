# 이커머스 크롤러 생성

이커머스팀 요청 크롤러 생성

상품 대표이미지, 상세 페이지 이미지, 네이버 쇼핑 정확도순 가격 5개 수집

## Installation

```
pip install pandas
pip install numpy
pip install selenium 
pip install beautifulsoup4
pip install chromedriver-autoinstaller

```
